# 색상 시스템 — 같은 톤 8색조 + 톤 사다리

> 디자인 원칙: **UI는 흑백 모노크롬(스위스 에디토리얼), 색은 오직 할일 막대(데이터)에만.**
> 흑백 그리드 위에서 색이 유일한 색이라 또렷이 도드라진다.

---

## 1. 모노크롬 셸 (UI 색)

| 토큰 | 값 | 용도 |
|---|---|---|
| `--bg` | `#FFFFFF` | 배경 |
| `--text` | `#141414` | 본문/날짜 |
| `--border` | `#8E8E8E` | 구조 하이라인 (상단바·탭 밑줄·리스트) |
| `--border-soft` | `#B6B6B6` | 보조 구분선 |
| `--muted` | `#777777` | 보조 텍스트 |
| `--faint` | `#A6A6A6` | 비활성/타 월 날짜·요일 라벨 |

오늘 표시 = 날짜 숫자에 **검은 원형**(`#141414`) 배경 + 흰 글자.

## 2. 프로젝트 색 — 같은 톤(S 58%, L 46%) · 다른 색조 8개

모든 프로젝트 색은 **동일 채도·명도**, 색조만 다르다 → 베이스 명도가 균일해 톤 사다리가 8색 모두 일관.

| # | 이름 | HEX | Hue |
|---|---|---|---|
| 1 | 레드 | `#B9313D` | 355 |
| 2 | 오렌지 | `#B97131` | 28 |
| 3 | 앰버 | `#B99E31` | 48 |
| 4 | 그린 | `#31B96A` | 145 |
| 5 | 틸 | `#31B5B9` | 182 |
| 6 | 블루 | `#3175B9` | 210 |
| 7 | 인디고 | `#5A31B9` | 258 |
| 8 | 퍼플 | `#B9319E` | 312 |

원색(위 HEX)은 **프로젝트 정체성** — 사이드바 범례·프로젝트 점/탭에 그대로 사용.

## 3. 태스크종류 톤 사다리 — 8단계 (전역 공유)

막대 색 = 프로젝트 원색에 톤 적용. 어두움→밝음 8단계. **3 dark(흰 글자) + 5 tint(검 글자)**, 중간 명도(베이스 근처) 사각지대는 의도적으로 건너뜀.

| 단계 | mode | k | 텍스트 | 비고 |
|---|---|---|---|---|
| L0 | `dark` | 0.50 | 흰 | 가장 진함 (시드 "마감") |
| L1 | `dark` | 0.43 | 흰 | |
| L2 | `dark` | 0.36 | 흰 | |
| L3 | `tint` | 0.32 | 검 | 시드 "작업" |
| L4 | `tint` | 0.44 | 검 | |
| L5 | `tint` | 0.56 | 검 | 시드 "회의" |
| L6 | `tint` | 0.68 | 검 | |
| L7 | `tint` | 0.82 | 검 | 가장 옅음 (시드 "리서치") |

```
dark:  out = base × (1 − k)
tint:  out = base × (1 − k) + 255 × k
막대 텍스트색 = 배경 명도 기준 흰/검 자동 (barText)
```

- **기본 시드 태스크종류 4개**(마감 L0 / 작업 L3 / 회의 L5 / 리서치 L7)는 사다리의 4개 단계에 매핑.
- 태스크종류는 **최대 8개**까지 각자 구분되는 톤을 가질 수 있다(권장 한계 = 8). 추가 시 미사용 단계를 우선 추천.

## 5. 날짜 마커 (이벤트 / 하드 데드라인) — 색이 아닌 흑백 기호

기간 막대(=할일)와 별개의 **점(날짜) 단위 표시**. 색상 막대와 구분되도록 흑백.

- **하드 데드라인**: 검은색 채운 칩 `⚑ 라벨` (`bg #141414`, 흰 글자)
- **이벤트**: 외곽선 칩 `◆ 라벨` (`border #141414`, 검은 글자)

// lib/color/compose.ts
export function applyTone(hex: string, t: { mode: "dark" | "tint"; k: number }): string {
  const rgb = hexToRgb(hex);
  const out = t.mode === "dark"
    ? rgb.map(c => c * (1 - t.k))
    : rgb.map(c => c * (1 - t.k) + 255 * t.k);
  return rgbToHex(out);
}

## 7. 메모

- 프로젝트 8색 + 태스크종류 8톤단계 동시 사용 가능(각 권장 한계 = 8). 더 필요하면 점/패턴 보조.
- 새 프로젝트 추가 시 색조는 미사용 hue 우선 추천하되 **S58 L46 톤 고정**. 새 태스크종류는 미사용 톤 단계 우선 추천.
- 텍스트 자동색은 조합별 계산이 정답(위 매트릭스가 근거).
